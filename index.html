How to setup nginx + php5-fpm on Debian Wheezy


In this article I will explain how to setup simple web server with nginx + php5-fpm on Debian Wheezy. 

The first part of tutorial will cover basic installation of nginx server and php5-fpm (FastCGI Process Manager - which is an alternative PHP FastCGI implementation). I decided to use the latest mainline version of nginx and php5-fpm package from official debian source. 

In the second part of this tutorial I will show you how to create simple web page which shows some basic PHP information from our system, and how to "connect" nginx with php5-fpm proccess manager and get all things to work.
